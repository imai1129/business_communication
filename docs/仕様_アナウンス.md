# お知らせ・メンテナンス 仕様

## 役割
- お知らせ・メンテナンスのタイトルと公開日付のみを一覧表示するセクション。

## 構成
- ソース: `components/dashboard/announcements.tsx`。
- レイアウト: 見出しと表示切替/並び順ボタン → フィルタボタン → 縦積みのリスト表示（1列）。
- UI部品: `Button`、`lucide`アイコン（`FileText` など）。

## データ
- `data/announcements.ts` のモック配列 `announcements` を import して使用。
  - フィールド: `id`, `title`, `date`, `image`, `content`。
  - 6件のサンプルレコード。

## 挙動
- 各アイテムはタイトル→公開日付の順で同一行に横並び表示（画像/本文/ダイアログなし）。
- 「リスト表示」「新着順」ボタンおよびフィルタボタン（ALL/新作情報/対象:SV向け/業務連絡（プロモ））はUIのみでロジックなし。

## 今後の検討メモ
- フィルタ/ソートを実装する場合、対象カテゴリの型定義を配列に追加。
- 実データ接続時はカードのリンク先（詳細ページ/外部URL）を持たせ、`Dialog` をリンク遷移に置換する選択肢も検討。
