# お知らせ・メンテナンス 仕様

## 役割
- お知らせ・メンテナンスのタイトルと公開日付を一覧表示し、クリックで詳細をダイアログ表示するセクション。カテゴリフィルタ（ALL/お知らせ/メンテナンス）で絞り込みできる。

## 構成
- ソース: `components/dashboard/announcements.tsx`。
- レイアウト: 見出しと表示切替/並び順ボタン → フィルタボタン → 縦積みのリスト表示（1列）。
- UI部品: `Button`、`Dialog`（Radixベース）、`next/image`、`lucide`アイコン（`FileText` など）。

## データ
- `data/announcements.ts` のモック配列 `announcements` を import して使用。
  - フィールド: `id`, `title`, `date`, `image`, `content`, `category` ("お知らせ"|"メンテナンス")。
  - 6件のサンプルレコード。

## 挙動
- 各アイテムをクリックするとダイアログが開き、タイトル・公開日付・（画像があれば）画像・本文を表示。
- TODO実施や店内共有のボタンは存在しない。
- フィルタボタン（ALL/お知らせ/メンテナンス）はロジック付きで、選択したカテゴリのデータのみを表示。新着順ボタンはUIのみでロジックなし。

## 今後の検討メモ
- フィルタ/ソートを実装する場合、対象カテゴリの型定義を配列に追加。
- 実データ接続時はカードのリンク先（詳細ページ/外部URL）を持たせ、`Dialog` をリンク遷移に置換する選択肢も検討。
