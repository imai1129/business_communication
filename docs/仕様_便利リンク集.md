# 便利リンク集 仕様

## 役割
- 右カラム下部の多段アコーディオンで、頻出リンクやカテゴリ別リンクを提示するセクション。

## 構成
- ソース: `components/dashboard/link-collection.tsx`。
- UI部品: Radix `Accordion`（`type="multiple"`）、`AccordionItem/Trigger/Content`、`lucide`アイコン（`Users`、`MoreHorizontal`、`List`）とプレーンテキストアイコン（お気に入りハート）。
- デフォルト開閉: `defaultValue` に `favorites` と `todo` を指定し、これらが初期展開。
- アイテム:
  - `favorites`（お気に入り）: サンプルリンク2件。
  - `hr`（人事・給与）: 「コンテンツなし」プレースホルダー。
  - `others`（その他）: 「コンテンツなし」プレースホルダー。
  - `todo`（ToDo）: サンプルリンク2件。
- スタイリング: 各アイテムは境界線・角丸付きの白背景カード風。Triggerに hover 無下線指定。

## データ/挙動
- `data/link-collection.ts` の `linkCategories` 配列を使用（id/label/items）。items は `[{id,label}]` としてリンク相当を保持。
- 完全静的。リンクURLは未設定で、サンプルテキストのみ。
- 折り畳み/展開はRadix Accordionのデフォルト動作任せ。

## 今後の検討メモ
- 実リンク化する場合、データ配列を用意しマッピングで描画・リンク先 (href) を付与。
- カテゴリ追加/並び順変更が必要なら配列化して管理しやすくする。
- ToDoが別機能と連携する場合、Badgeや件数表示をTriggerに追加する選択肢。
